# Installation

## From pypi

QEnergy can be installed from PyPi with the following command:

```{prompt} bash

pip install qenergy
```

## From source

The software can be downloaded from https://github.com/RajaYehia/QEnergy, using for instance git

```{prompt} bash

git clone https://github.com/RajaYehia/QEnergy
```

and installed with poetry 

```{prompt} bash

cd QEnergy
poetry install
```

or pip

```{prompt} bash

cd QEnergy
pip install .
```

It is also possible to install it directly from GitHub with pip:

```{prompt} bash

pip install git+https://github.com/RajaYehia/QEnergy
```
